# 数据说明

本正式版仓库不直接包含完整原始数据和大型中间数据，仅保留论文展示、代码结构和关键结果所需的说明文件。

## 1. 数据类型

项目中使用的数据主要包括：

- 科创板股票基础信息；
- 科创 50 指数成分与权重；
- 股票日行情与日收益率；
- 估值、财务和交易类指标；
- 多因子特征面板；
- 股票关系图节点与边数据；
- 回测所需的预测值、权重和策略收益。

## 2. 为什么不直接上传完整数据

- GitHub 对单文件大小有限制；
- 原始 CSV 和处理后数据体积较大；
- 部分数据可能来自第三方数据库或特定研究环境；
- 正式展示版应保持简洁，避免把大型数据仓库化。

## 3. 复现时的数据放置建议

如果需要复现代码，建议在本地按照如下结构准备数据：

```text
data/
  raw/          原始行情、财务、指数权重等数据
  processed/    清洗后的市场数据、因子数据和图结构快照
  external/     如有第三方补充数据，可放在此处
```

然后根据 `src/gl_v4/` 中脚本的数据路径进行调整。
