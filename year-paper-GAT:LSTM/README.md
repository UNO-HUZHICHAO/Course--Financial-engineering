# 科创 50 指数增强策略：弱信号环境下的图网络机制诊断与改进

本仓库为学年论文项目的正式 GitHub 展示版，主要整理了论文最终稿、核心图表、最终模型代码、关键回测结果和必要说明文档。仓库已经去除了历史旧版本、早期尝试、大量中间结果和不适合直接上传的大型原始数据文件，结构更适合作为正式项目展示与后续复现参考。

## 1. 项目简介

本项目围绕科创 50 指数增强策略展开，研究在 A 股科创板弱信号环境下，如何结合多因子模型、股票关系图、图神经网络和组合优化方法，构建具有解释性和一定超额收益能力的指数增强框架。

项目重点关注以下问题：

- 如何从行情、估值、交易和波动等维度构造多因子特征；
- 如何利用股票之间的行业、相关性或潜在关系构建图结构；
- 图神经网络是否能够在弱信号环境下提供增量信息；
- GAT、LSTM 和融合机制在指数增强任务中的表现差异；
- 在预测信号较弱时，组合优化约束如何影响最终回测表现；
- 如何通过因子 IC、分组收益、消融实验和净值曲线诊断模型有效性。


## 6. 复现说明

本仓库更偏向正式展示和结果归档，而不是一键全量复现。若要复现实证流程，建议按以下顺序：

1. 阅读 `docs/project_framework_details.md` 和 `docs/modeling_algorithm_details.md`；

6. 对比 `results/final_backtest/` 中的结果文件。
