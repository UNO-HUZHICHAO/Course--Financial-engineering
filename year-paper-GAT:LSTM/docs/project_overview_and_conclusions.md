# 科创50指数增强策略 — 整体思路与最终结论

> **项目**：基于双流门控图网络的科创板50指数增强策略研究
> **最终版本**：GL V4
> **运行环境**：AutoDL RTX 5090 (32GB) / PyTorch 2.7.0 / CUDA 12.8
> **日期**：2026年6月1日

---

## 一、研究目标

本研究以上证科创板50成份指数（000688.SH）为基准，构建一套**双流门控图神经网络（Dual-Stream Gated Graph Network）**选股模型，目标是在控制跟踪误差的前提下，通过融合**量价时序信号**与**产业链图谱结构信号**，获取相对于基准的正向超额收益（Alpha）。

回测区间：2020-07-01 至 2025-12-31，覆盖科创板约600只股票。

## 二、核心术语释义

为便于阅读，以下对文档中频繁出现的英文术语做统一中文释义：

### 2.1 模型架构术语

| 英文术语                                         | 中文释义                                                                                                                                                                                                                                       |
| :----------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **LSTM**（Long Short-Term Memory）         | 长短期记忆网络。一种循环神经网络，负责从每只股票最近20个交易日的量价因子序列中提取**时序特征**。本模型中作为"时序流"（Temporal Stream）。                                                                                                |
| **GAT**（Graph Attention Network）         | 图注意力网络。一种图神经网络，负责从股票间关系图（产业链图谱）中提取**空间结构特征**。本模型中作为"图流"（Spatial Stream）。                                                                                                             |
| **Dual-Stream**（双流）                    | 模型同时拥有LSTM和GAT两条独立的特征提取通路，分别处理时序信息和图结构信息。                                                                                                                                                                    |
| **GatedFusion**（门控融合）                | 早期版本的融合方式。学习一个标量门控值$g \in [0,1]$，最终输出 = $g \times$ LSTM特征 + $(1-g) \times$ GAT特征。$g$ 越接近1表示越依赖时序信号。                                                                                          |
| **FiLM**（Feature-wise Linear Modulation） | 特征级线性调制。与门控融合不同，FiLM让GAT输出一对调制参数$(\gamma, \beta)$，直接对LSTM特征做仿射变换：$H_{new} = \gamma \cdot H_{LSTM} + \beta$。GAT不再是"并列选手"，而是"背景调制器"——它调节LSTM对不同特征的敏感度，而非直接参与打分。 |
| **AdaptiveFiLM**（自适应FiLM）             | 在FiLM基础上，模型根据每个样本中LSTM与GAT预测方向的一致性、强度比例等信息，动态学习一个"信任分数"（trust score）。当两个流方向一致时放大调制，方向冲突时自动收缩，降低图信号对时序信号的干扰。                                                 |

### 2.2 组合构建术语

| 英文术语                                                  | 中文释义                                                                                                                                                                                      |
| :-------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **layered_equal_weight**（分层等权）                | 本研究的**主推荐策略**。将模型打分排名前60的股票分为三层：Top 20 各配0.20%、第21~40名各配0.175%、第41~60名各配0.125%。最终权重 = 90%基准权重 + 10%分层主动权重。每10个交易日调仓一次。 |
| **qp_guarded**（QP保护型优化）                      | 使用CVXPY二次规划求解器的组合构建方式，带有行业偏离、个股权重上限等多级约束。理论上更"精确"，但在弱信号环境下容易被极端分数主导，产生集中持仓和高换手。                                       |
| **adaptive_layered_equal_weight**（自适应分层等权） | 在分层等权基础上引入自适应权重分配，根据策略间的历史表现动态调整层间配比。                                                                                                                    |

### 2.3 消融实验术语

| 英文术语                       | 中文释义                                                                               |
| :----------------------------- | :------------------------------------------------------------------------------------- |
| **full**                 | 完整双流模型，LSTM + GAT + 融合层全部启用。                                            |
| **lstm_only**            | 仅启用LSTM时序流，强制关闭GAT（门控值固定为1.0）。用于验证"图结构是否带来了额外信息"。 |
| **gat_only**             | 仅启用GAT图流，强制关闭LSTM（门控值固定为0.0）。用于验证"图的独立选股能力"。           |
| **ablation**（消融实验） | 逐步关闭模型某个组件，观察性能变化，以判断该组件的真实贡献。                           |

### 2.4 其他关键术语

| 英文术语                                    | 中文释义                                                                                              |
| :------------------------------------------ | :---------------------------------------------------------------------------------------------------- |
| **Alpha**（超额收益）                 | 策略收益率减去基准收益率的部分，衡量策略的选股能力。正值表示跑赢基准。                                |
| **IR**（Information Ratio，信息比率） | Alpha / 超额收益标准差，衡量单位主动风险下的超额回报，类似夏普比率但针对超额部分。                    |
| **Rank IC**（排名相关系数）           | 模型预测分数与未来实际收益的Spearman秩相关系数。正值表示模型有一定排序预测能力。                      |
| **OOS**（Out-of-Sample，样本外）      | 未参与模型训练的数据区间。所有报告中的收益指标均为OOS结果，确保评估公平性。                           |
| **Anchor**（锚点）                    | 块状滚动回测中的重训时间点。每个锚点使用此前所有历史训练模型，在之后的OOS区间评估。本研究共10个锚点。 |
| **Early Stopping**（早停）            | 当验证集损失连续5轮不再改善时自动停止训练，防止过拟合。                                               |
| **Edge Profile**（边配置）            | 图网络中保留哪些类型的关系边。例如 `no_industry` 表示剔除行业关联边。                               |
| **Snapshot**（快照）                  | 某个时间切片的图结构数据，包含节点特征和边关系。每季度更新一次。                                      |

## 三、模型整体架构

```
                    ┌─────────────────────────────────────────┐
                    │           科创50指数增强模型              │
                    └─────────────────────────────────────────┘

  ┌──────────────────────────┐    ┌──────────────────────────────┐
  │    时序流 (LSTM)          │    │     图流 (GAT)               │
  │                          │    │                              │
  │  每只股票最近20个交易日    │    │  季度产业链图谱快照           │
  │  的量价因子序列            │    │  (股权/客户/供应商/LLM边)    │
  │  [B, 20, F] → LSTM       │    │  节点16维特征 → 双层GAT      │
  │  → H_LSTM [B, 64]        │    │  → H_GAT [N, 64]            │
  └──────────┬───────────────┘    └──────────┬───────────────────┘
             │                               │
             └──────────┬────────────────────┘
                        ▼
              ┌─────────────────────┐
              │   融合层 (FiLM)      │
              │                     │
              │  γ, β = GAT调制参数  │
              │  H = γ·LSTM + β     │
              └─────────┬───────────┘
                        ▼
              ┌─────────────────────┐
              │  预测头 (Linear)     │
              │  Score ∈ ℝ          │
              └─────────┬───────────┘
                        ▼
              ┌─────────────────────┐
              │  组合构建             │
              │  分层等权 / QP优化   │
              │  → 最终持仓权重      │
              └─────────────────────┘
```

**训练方式**：块状滚动训练（Expanding Window），共10个锚点，每个锚点从头训练模型，验证集做早停（patience=5），然后在锚点之后的OOS区间评估。

**损失函数**：$L = 0.7 \times (1 - \text{RankCorr}) + 0.3 \times \text{MSE} + 0.02 \times \text{Gate正则}$

## 四、版本演进

### 4.1 GL V1：初版验证 — 路线证伪

**核心思路**：

- 搭建LSTM + GAT双流框架原型
- GAT节点仅4维结构特征（研发强度、技术护城河、高新资质、小巨人资质）
- 使用QP优化器构建组合

**关键发现**：

- 年化Alpha **-10.88%**，策略完全失败
- 诊断发现：GAT因图谱极度稀疏（早期快照仅1条边）而退化为逐节点MLP
- QP优化器在弱信号下严重放大噪声，产生集中持仓和高换手

**结论**：模型框架方向正确，但**组合构建层是最大瓶颈**——弱信号无法支撑精确优化。

---

### 4.2 GL V1 + 分层等权 — 关键转折

**核心思路**：

- 保持模型不变，**仅替换组合构建方式**
- 从QP优化切换为分层等权（layered_equal_weight）

**关键发现**：

- 年化Alpha从 -10.88% 翻转为 **+0.64%**
- 同样的模型打分，仅因组合方式不同，收益差距超过11个百分点

**结论**：**组合构建方式比模型架构更重要**。弱信号环境下，简单的分层排名配置远优于精确但脆弱的QP优化。分层等权的核心优势在于不区分排名21和排名22的股票——这种"模糊正确"比"精确错误"更稳健。

---

### 4.3 GL V2：因子增强与季度快照

**核心思路**：

- GAT节点特征从4维扩展到**9维**（4维结构 + 5维季度财务因子）
- 图快照从半年更新升级为**季度更新**（14个快照期）
- 引入门控防坍缩机制（gate floor=[0.20, 0.80]），防止门控值退化到极端
- 损失函数改为组合损失：70%排序相关 + 30%MSE + 门控正则
- 调仓频率从日频改为**周频**（5个交易日），减少超高换手率带来的过高交易成本

**关键发现**：

- 5维财务因子的边际贡献极小甚至为负：无财务Alpha 0.83% vs. 有财务Alpha 0.80%
- 但整体框架更稳定，门控机制工作正常
- 年化Alpha：**+0.83%**（分层等权）

**结论**：季度快照和门控防坍缩是有效改进。财务因子未带来显著增量，可能因为科创板公司的财务数据披露质量参差不齐，或因子本身已被量价信号间接覆盖。V2确立为正式基线。

---

### 4.4 GL V3 Enhanced：全面扩容

**核心思路**：

- **时序因子**从16维扩展到**33维**（base16 + micro8 + adv9），引入微观结构因子和高阶统计因子
- **图节点特征**扩展到**16维**（4结构 + 5财务 + 3基本面动量 + 4估值）
- **标签平滑**：从T+1单日收益改为**未来5日平均收益**，降低噪声约 $\sqrt{5}$ 倍
- 支持消融实验（full / lstm_only / gat_only）
- 引入因子正交化去除共线性

**关键发现（正式回测结果）**：

| 消融模式  | 因子组        | Alpha（分层等权） |
| :-------- | :------------ | :---------------: |
| full      | base16+micro8 |      +0.42%      |
| full      | all33         |      +0.46%      |
| lstm_only | base16+micro8 | **+0.65%** |
| gat_only  | base16+micro8 | **-0.01%** |

**结论**：GAT图流**未能带来稳定增量**。纯LSTM反而表现最好，GAT-only接近零收益。这说明V3的GAT支路虽然信息更丰富了，但融合机制没有有效地将图信号转化为选股增益。问题可能出在图的边结构上，而不是节点特征上。

---

### 4.5 GL V4：双流抢救 — 最终版本

**核心思路**：
GL V4不是继续堆模型复杂度，而是系统性地诊断和修复GAT支路的问题。

#### 第一步：图谱瘦身（边消融）

创建4种边配置（edge profile），逐一验证哪类边是噪声源：

| 边配置          | 保留的边类型                      | 结论                         |
| :-------------- | :-------------------------------- | :--------------------------- |
| `full`        | 股权 + 客户 + 供应商 + 行业 + LLM | 全量边，基线                 |
| `hard`        | 仅股权 + 客户 + 供应商            | 硬边图                       |
| `no_llm`      | 股权 + 客户 + 供应商 + 行业       | 剔除LLM抽取边                |
| `no_industry` | 股权 + 客户 + 供应商 + LLM        | **剔除行业边（最佳）** |

**关键发现**：**行业边是主要噪声源**。行业边按申万一级行业全连接构建，权重固定0.2，造成图过度密集和同质化，使GAT表征趋于坍缩（所有节点的图嵌入变得几乎一样）。`no_industry` 配置下GAT表征区分度显著提升。

#### 第二步：融合方式改造（从Gate到FiLM）

将GAT从"并列选手"改为"背景调制器"：

| 融合方式          | 原理                               | 效果                              |
| :---------------- | :--------------------------------- | :-------------------------------- |
| GatedFusion（旧） | $g \cdot LSTM + (1-g) \cdot GAT$ | GAT经常抢权重但信号弱时反噬       |
| FiLM              | $\gamma \cdot LSTM + \beta$      | GAT不直接打分，而是调节LSTM敏感度 |
| AdaptiveFiLM      | 根据方向一致性动态调整调制强度     | 方向冲突时自动收缩                |

**关键发现**：FiLM在"强窗口"（GAT信号好的时段）表现优异，但在"弱窗口"（GAT信号差的时段）可能拖累LSTM。这说明GAT不是完全无用，而是"有时有用，有时有害"。

#### 第三步：保护机制与稳健化

- 保护型FiLM（GuardedFiLM）：当LSTM与GAT方向冲突时自动关闭调制
- 最终锁定主线配置：`ablation=full, factor-set=base16+micro8, graph-profile=no_industry, fusion-mode=adaptive_film, qp-profile=guarded`

## 七、最终结论

### 结论一：三个策略在两个种子下均实现正Alpha

分层等权（layered_equal_weight）作为主推荐策略，在seed=42下Alpha为+1.42%，在seed=123下Alpha为+1.18%，均为正值且方向一致。这意味着模型具备稳定的选股排序能力，不是随机波动的结果。

### 结论二：分层等权始终优于QP优化

在所有种子下，分层等权的Alpha均高于QP保护型优化。这再次验证了核心洞察：**在弱信号环境下，"模糊正确"的分层排名配置优于"精确脆弱"的数学优化**。QP优化器虽然理论上更精确，但容易被模型预测分数中的噪声放大为集中持仓和高换手，最终侵蚀收益。

### 结论三：Seed 42 表现略优于 Seed 123

Seed 42在所有策略上的Alpha均高于Seed 123约0.2~0.3个百分点，信息比率也更高（0.57~0.65 vs. 0.46~0.50）。但两者差距在合理范围内，不构成对模型稳健性的质疑。

### 结论四：图结构（GAT）并非完全无效

从V1的-10.88%到V4的+1.42%，经过图谱瘦身（剔除行业边）、融合方式改造（FiLM替代Gate）、以及组合构建优化（分层等权替代QP），双流模型实现了显著改善。特别是：

- **行业边是主要噪声源**：剔除后GAT表征区分度显著提升
- **FiLM比Gate更合理**：让GAT做背景调制而非并列打分，更符合图信号低频、慢变的特性
- **但GAT的贡献仍有窗口依赖性**：在某些时间窗口有效，某些窗口可能拖累

### 结论五：建议扩大到更多种子验证

当前两种子结果方向一致，说明模型具有初步鲁棒性。建议后续扩展到5~8个种子（如7、21、256、512、1024等），构建Alpha的均值和置信区间，进一步验证策略的统计显著性。

### 结论六：版本演进的核心收获

| 版本                 | 核心改进                                      |        Alpha变化        |
| :------------------- | :-------------------------------------------- | :---------------------: |
| V1                   | 原型搭建 + QP优化                             |         -10.88%         |
| V1+分层等权          | 仅改组合构建                                  |         +0.64%         |
| V2                   | 季度快照 + 门控防坍缩 + 周频调仓              |         +0.83%         |
| V3                   | 33因子 + 标签平滑 + 消融框架                  |      +0.42%~+0.65%      |
| **V4（最终）** | **图谱瘦身 + FiLM融合 + adaptive_film** | **+1.18%~+1.42%** |

从V1到V4，年化Alpha从-10.88%提升到+1.42%，改善幅度超过12个百分点。其中**最大的单一贡献来自组合构建方式的改变**（QP→分层等权，贡献约+11个百分点），其次是**图谱瘦身和融合方式改造**（V2→V4，贡献约+0.6个百分点）。

## 九、局限性与后续方向

1. **种子数量有限**：当前仅验证了2个种子，统计显著性待更多种子确认
2. **GAT贡献的窗口依赖性**：图信号在某些时段有效、某些时段拖累，尚未找到完美的动态切换机制
3. **因子体系可进一步优化**：V3曾试验33因子但效果不如24因子（base16+micro8），说明因子并非越多越好，后续可做更精细的因子筛选
4. **标签与调仓频率的匹配**：当前使用5日平均收益标签但10日调仓，存在一定错配
5. **异质图算子**：当前GAT默认同质性假设，未来可探索MixHop、H2GCN等异质图专用算子

---

> **总结**：本研究经过V1→V2→V3→V4四个版本的迭代演进，从初始的-10.88% Alpha提升至+1.42%。核心收获是：在弱信号选股场景下，**组合构建方式的稳健性**远比模型架构的精巧更重要；**图谱结构信号不是无效，而是需要精细的噪声治理**（剔除行业边）和**合理的融合方式**（FiLM调制优于门控融合）。
