## 事件简介 
曾是中国零售巨头的苏宁易购，在2015-2019年间进行了激进的跨界并购（如收购天天快递、家乐福中国、国际米兰足球俱乐部等）。这种“短债长投”的模式极大地消耗了公司的现金流。
2020年底，受宏观环境和债务集中到期影响，苏宁突发百亿债券危机，流动性瞬间枯竭，导致其主体信用评级被大幅下调，最终不得不引入国资进行债务重组。本案例经典地展示了**盲目扩张如何演变为致命的信用风险**。

---

## 整体分析框架 
1. **是什么 (What)**：苏宁的扩张史与 2020 年底的债务违约表象。
2. **为什么 (Why)**：资产负债表恶化，短期刚性债务过高，流动性管理失败。
3. **怎么预测 (How/Quant)**：利用 KMV 结构化模型，结合真实 A 股市场数据，测算苏宁在暴雷前夕的违约距离(DD)与预期违约概率(EDF)，验证量化风控的前瞻性。
4. **怎么办 (Action)**：企业应对集中度风险、流动性缺口的防范措施。
